# [🚀Micrositio de un evento científico](https://juanmx.github.io/micrositio-evento-cientifico)

Un investigador que conozco me contactó para preguntarme si podía hacer un micrositio para un evento científico que se llevaría a cabo a mitad del año 2026.

Dije que si y el cliente me envió las especificaciones de lo que necesitarían para la página. En resumen, querían un página minimalista y responsiva. Yo hice el micrositio con "Bootstrap" y "FontAwesome".

Ellos querían una "página base" para después agregarle los colores, imágenes y tipos de letra institucionales. **Esa "página base" es lo que está en este repositorio**. Solicité subir a un repositorio público la "página base" y ellos accedieron. 

**De manera creativa agregué un imagen gif como anuncio, pues ando buscando trabajo.**

Según [mailchimp](https://mailchimp.com/es/resources/what-is-a-microsite/) un micrositio es: Sitios web pequeños o páginas web simples que utilizan un dominio o subdominio diferente, lo que los separa del sitio web principal de la empresa. En última instancia, un micrositio actúa como un sitio web separado para una sola empresa, pero puede transmitir un mensaje único que difiere del sitio web habitual de la empresa. Los micrositios suelen ser temporales, pero pueden usarse a largo plazo, dependiendo de los fines.

A researcher I know contacted me to ask if I could create a microsite for a scientific event taking place in the middle of 2026.

I agreed, and the client sent me the specifications. In short, they wanted a minimalist and responsive website. I made the microsite using "Bootstrap" and "Font Awesome".

They wanted a "base page" to add later their institutional colours, images, and fonts. **That "base page" is in this repository**. I requested to upload the "base page" to a public repository and they agreed. 

**In a creative way, I added a gif image as an ad because I'm looking for a job.**

According to [Mailchimp](https://mailchimp.com/resources/what-is-a-microsite/), a microsite is: Small websites or single web pages that use a different domain or subdomain, separating them from the company's main website. Ultimately, a microsite acts as a separate website for a single company, but it may convey a unique message that differs from the business' everyday website. Microsites are typically temporary but can be used long-term, depending on their purposes.
